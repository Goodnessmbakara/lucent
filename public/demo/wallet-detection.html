How Lucent Will Use This

🎯 Step 1: Detection

Lucent detects which wallet is installed by checking for injected global objects. No user action required.

🪝 Step 2: Hook Transaction Requests

When a dApp initiates a transaction, Lucent intercepts it before the wallet sees it.

// Lucent's interception approach:
const originalRequest = window.StacksProvider.request;

window.StacksProvider.request = async (method, params) => {
  if (method === 'stx_signTransaction') {
    // Show Lucent preview popup here!
    const approved = await showLucentPreview(params);
    if (!approved) return; // User rejected
  }
  // Pass through to actual wallet
  return originalRequest(method, params);
};

✅ Step 3: Display Preview & Security Checks

Lucent decodes the transaction, runs security checks, and shows the user a clear preview before they approve in their wallet.

🔐 Key Advantage

No wallet modifications needed. This works with any Stacks wallet that follows the standard provider injection pattern. Wallets never even know Lucent exists.
